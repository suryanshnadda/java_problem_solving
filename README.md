# java_problem_solving

This repository contains a curated collection of Java problem-solving programs.
The focus is on strengthening core Java fundamentals and algorithmic thinking.
All solutions are written with clarity, correctness, and proper edge-case handling.
The code is suitable for interview preparation and academic practice.

---

## Topics Covered

### Array Problems
1. Second largest element of an array (sorted array).
2. Second largest element of an array (unsorted array).
3. Second smallest element of an array (sorted array).
4. Second smallest element of an array (unsorted array).
5. Check whether an array is sorted.
6. Remove duplicate elements from an array.
7. Move array elements left by one place.
8. Move array elements right by one place.
